⚙️ Connect Live Data
Paste your Google Apps Script Web App URL below. This is a one-time setup — the URL is saved in your browser and the portal fetches fresh data automatically every time it opens.
Step-by-step setup:
1. Go to script.google.com → click New project
2. Delete all default code → paste the full apps_script_lld.js code
3. Click 💾 Save → then Deploy → New deployment
4. Click the ⚙️ gear next to "Select type" → choose Web app
5. Execute as: Me · Who has access: Anyone (not "Anyone with Google account")
6. Click Deploy → click Authorize access → Allow
7. Copy the Web app URL (starts with https://script.google.com/macros/s/)
8. Paste it below ↓
